# Spectrum Outfitters Launcher

Desktop launcher application for easy start/stop control of the Spectrum Outfitters application.

## Usage

**Windows:** Double-click `start-launcher.bat` or `START_APPLICATION.bat` from the root directory.

This will:
1. Install launcher dependencies if needed
2. Start the launcher web interface
3. Open your browser to the launcher control panel

## Features

- ✅ One-click start/stop
- ✅ Real-time status monitoring
- ✅ Direct links to application and API
- ✅ Clean, modern web interface

## Port

The launcher runs on port **3001** by default. Make sure this port is available.

## Manual Control

If you prefer command-line control, use `START_APPLICATION.bat` in the root directory to start everything with one click.
