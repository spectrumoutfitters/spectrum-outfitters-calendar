# Spectrum Outfitters logo assets

Place your logo files here so the app and "Add to Home Screen" use the correct branding:

- **spectrum-icon.png** — Use the **Spectrum Outfitters Emblem Icon** (gold S + mountains in a circle).  
  Used as the favicon and as the **iPhone/iPad "Add to Home Screen" icon**.  
  **For iPhone:** Export this as **180×180 pixels** (or larger) so the home screen icon looks sharp. If the icon is smaller, iOS may scale it up and it can look blurry or not appear.

- **spectrum-logo.png** — Use the **Spectrum Outfitters Full Logo** (SPECTRUM OUTFITTERS + tagline).  
  Used in the sidebar, login, and header.

If these files are missing, the app will fall back to existing logo.svg/logo.png or show the "SO" placeholder.

**If the iPhone home screen still shows the wrong icon:** Remove the shortcut from the home screen, then add it again (Share → Add to Home Screen). iOS caches the icon heavily.
